很爛的中文 Spigot 1.20+ 插件</h3>
<h1>指令 - Commands:</h1>
<h1>/lore [文字] </h1>
<h3>權限: 無</h3>
<h3>功能: 為你手上的物品新增註解, 支援"&"顏色代碼 (不是每種物品都能, 部分不能添加lore)</h3>
<h3>Lore要換行的話 重打一次指令,輸入第二行、第三行...等的文字 </h3>
<h1>/miniwarn [玩家] </h1>
<h3>權限: miniblack.miniwarn</h3>
<h3>功能: 非常明顯的警告玩家</h3>

<h3>權限: miniblack.magicstick</h3>
